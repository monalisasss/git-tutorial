Passo a passo - Criar repositório

  1. Criar repositório do github https://github.com/
  2. git statusno terminal
  3. git add .para adicionar todos os arquivos da pasta
  4. git remote add origin https://github.com/usuario/nome-do-repositorio.gitpara linkar pasta com repositório
  5. git statusconfirmar se arquivos estão em verde ou seja, se foram adicionados
  6. git commit -m "initial commit"inserir mensagem do commit
  7. git pullopcional, quando estamos trabalhando em conjunto
  8. git push origin masterenviar para a master
  9. fazer loginfora do terminal
  10. conferir se os arquivos aparecem no githubhttps://github.com/usuario/nome-do-repositorio.git
